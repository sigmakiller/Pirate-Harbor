# ATLAS OS Design System

## Vision

Atlas is a premium desktop operating system for preserving a player's
gaming history---not a traditional launcher.

## Principles

-   Monochrome operating system
-   Typography is the identity
-   Games are the content
-   Whitespace is intentional
-   Contextual immersion only on Game Detail pages

## OS Colors

Background: #050505 Surface: #101010 Cards: #171717 Text: #F5F5F5
Secondary: #A1A1A1 Borders: rgba(255,255,255,.08)

## Layout

12-column grid 72px margins 32px gutters 128px vertical spacing

## Typography

Inter JetBrains Mono Space Grotesk

## Golden Rule

Atlas OS is the museum. The game page is the exhibition. The museum
never changes.
